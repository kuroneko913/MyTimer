# MyTimer
Chrome extension for practice.

images : Getting Started Tutorial https://developer.chrome.com/extensions/getstarted

アイコンをクリックすると、インターバルタイマーとアラームが起動する

インターバルタイマー
- 90分経過するたびに、お知らせする。

タイマー
- 1,5,10,30,60,90分を計測し、お知らせするタイマー
-  右メニュークリックで使用可能

アラーム
- 指定した時刻になると、お知らせするタイマー
- 指定したページを新しく開く機能付き
